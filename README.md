Контрольная работа

Задание 1.1: Вычисление интеграла с использованием потоков

Описание
Реализация вычисления интеграла функции sin(x) от 0 до 1 методом прямоугольников с использованием многопоточности и механизмов синхронизации.

Функциональность
1. Класс IntegralCalculator - вычисление интеграла с событиями для прогресса и результатов
2. Многопоточность - запуск вычислений в отдельных потоках
3. Приоритеты потоков - сравнение работы потоков с Highest и Lowest приоритетами
4. Синхронизация:
   - Lock (только один поток)
   - SemaphoreSlim (ограничение количества одновременно работающих потоков)
5. Прогресс выполнения - вывод прогресса в консоль через события

Структура проекта
- `IntegralCalculatorLib/` - библиотека классов с основным функционалом
- `IntegralApp/` - консольное приложение для демонстрации работы

Как запустить
1. Открыть решение `ControlWork1.sln` в Visual Studio
2. Установить проект `IntegralApp` как запускаемый
3. F5 для запуска

Результат выполнения
Программа выводит:
- Прогресс вычислений (0-100%)
- Результат интеграла и время выполнения
- Сравнение потоков с разными приоритетами
- Тесты синхронизации (lock и semaphore)

Технологии
- C# .NET
- System.Threading
- System.Threading.Tasks
- Events для оповещения о прогрессе
- Stopwatch для измерения времени
